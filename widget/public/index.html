Botla Widget

Bu alan, Botla müşterilerinin web sitelerine entegre ettiği chatbot widget'ını barındırmak için kullanılmaktadır.

Nasıl Çalışır?

Botla ile oluşturduğunuz yapay zeka chatbot'unuz, tek satır JavaScript kodu ile web sitenize eklenir. Widget, bu alan üzerinden yüklenir ve sitenizde görüntülenir.
